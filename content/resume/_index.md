    - 负责河道“四乱”矢量数据的清洗、入库与空间分析
    - 利用成果数据制作每一期成果报告里面的专题图。
  - **关键贡献/成果**：
    - 利用 Python + GDAL 实现“四乱”成果矢量数据**完整性检查**的自动化脚本工具，审核效率提高**200%**，入库成功率达到**100%**。
    - 利用 Python + ArcPy 实现“四乱”成果专题图的自动化生成，制图效率提升**600%**（原本6天的工作，一天就完成）。

#### 2）🌬️ 东莞市"松山湖大气遥感监测"服务项目（2021.11-2022.03）
*主导构建松山湖区域大气污染物短时预测模型，利用遥感与监测数据，开发从 **数据处理 → 时空预测 → GIS 可视化集成** 的完整技术链路，为管委会提供未来数小时空气质量预报服务。*

  - **角色**：GIS算法工程师
  - **背景**：整合园区监测与遥感数据，搭建小时级短时空气质量时空预测与可视化预报系统，支持管委会环境管理与应急决策。
  - **任务**：
    - 收集和清洗项目中每日生产的松山湖历史大气监测数据，构建具有时空网格特征的数据集。
    - 设计并实现时空预测模型框架，完成空气污染物（PM₂.₅、PM₁₀ 等）的短时预测。
